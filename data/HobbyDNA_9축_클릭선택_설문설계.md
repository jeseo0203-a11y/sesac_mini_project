# 하비DNA — 취미 프로파일 벡터 9축 클릭 선택형 설문 설계 v1.0

원본 기획서 「2. 무엇을 받아들이나 — 취미 프로파일 벡터」의 9개 차원을,
대화형(LLM 자유응답)이 아닌 **클릭 선택(버튼/칩) UI**로 구현하기 위한 스펙입니다.
설문 피로를 줄이기 위해 **핵심 5개 → 보강 4개** 순서의 점진 프로파일링 구조를 적용했습니다.

---

## 📌 진행 순서 제안

| 단계 | 목적 | 포함 축 |
|---|---|---|
| **1단계 (핵심 5개, 첫 화면)** | 실현가능성을 빠르게 좁힘 — 여기까지만 답해도 1차 추천 가능 | 에너지·번아웃 / 사회성 / 활동량 / 시간·낮밤 / 예산 |
| **2단계 (보강 4개, 이후 점진 노출)** | 취향을 정교화 — 추천 카드를 보다가, 또는 "더 정확하게 찾기" 버튼을 눌렀을 때 노출 | 동기(목적) / 성취 방식 / 공간·오감 / 기질·신규성 |

1단계만으로도 "회복 우선 모드 vs 발산 모드" 같은 큰 방향은 잡히고, 2단계가 채워질수록 취미 DNA 리포트의 설명가능성(왜 이 취미인지)이 정교해지는 구조입니다.

---

## 1️⃣ 에너지·번아웃 *(핵심)*

**질문 문구**: "요즘 컨디션은 어떤가요?"
**UI 타입**: 단일 선택 (4개 카드, 세로 배치 + 이모지)

| 클릭 옵션 | 내부 태그 | 벡터 값 |
|---|---|---|
| 😮‍💨 완전히 방전됐어요 | `burnout_severe` | energy = -2 |
| 😔 좀 지쳐있어요 | `burnout_mid` | energy = -1 |
| 🙂 그럭저럭 괜찮아요 | `neutral` | energy = 0 |
| ⚡ 에너지가 넘쳐요 | `energetic` | energy = +2 |

**추천 영향**: 추천 '강도' 조절 (회복형↔발산형). `burnout_severe` 선택 시 자동으로 초저강도·단기완료·혼자형 취미만 상위 노출 + 안전문구("괜찮아요, 오늘은 가볍게 시작해요") 동반.

---

## 2️⃣ 사회성 *(핵심)*

**질문 문구**: "혼자가 편할까요, 함께가 좋을까요?"
**UI 타입**: 5점 클릭 스케일 (가로 배치, 슬라이더처럼 보이는 버튼 그룹)

| 클릭 옵션 | 내부 태그 | 벡터 값 |
|---|---|---|
| 완전 혼자가 좋아요 | `solo_strong` | sociality = -2 |
| 혼자가 편해요 | `solo_mild` | sociality = -1 |
| 둘 다 괜찮아요 | `flexible` | sociality = 0 |
| 함께가 좋아요 | `social_mild` | sociality = +1 |
| 사람들과 왁자지껄이 좋아요 | `social_strong` | sociality = +2 |

**추천 영향**: 혼자형/모임형 분기. 이후 "함께" 계열 선택 시 문토·소모임 핸드오프 가중치 ↑.

## 3️⃣ 활동량 *(핵심)*

**질문 문구**: "몸은 얼마나 움직이고 싶으세요?"
**UI 타입**: 단일 선택 (4개 카드)

| 클릭 옵션 | 내부 태그 | 벡터 값 |
|---|---|---|
| 🪑 거의 안 움직이는 게 좋아요 | `static` | activity = 0 |
| 🚶 가볍게 몸 푸는 정도 | `light` | activity = 1 |
| 🏃 땀나게 움직이고 싶어요 | `moderate` | activity = 2 |
| 🔥 숨차게 격렬한 것도 좋아요 | `intense` | activity = 3 |

**추천 영향**: 신체 강도 매칭 (필사·독서 ↔ 클라이밍·러닝).

---

## 4️⃣ 시간·낮밤 *(핵심)*

**질문 문구 A**: "언제 시간이 나세요?" — 다중 선택 가능
**질문 문구 B**: "몇 시쯤 제일 활기가 도세요?" — 단일 선택

| 질문 A 옵션 | 내부 태그 |
|---|---|
| 평일 저녁 | `weekday_evening` |
| 평일 낮 (재택·프리랜서 등) | `weekday_day` |
| 주말 | `weekend` |

| 질문 B 옵션 | 내부 태그 |
|---|---|
| 🌅 아침형 | `morning` |
| ☀️ 낮 시간대 | `daytime` |
| 🌆 저녁형 | `evening` |
| 🌙 완전 야행성 | `night_owl` |

**추천 영향**: 실현가능성·시간대 추천. 예: `night_owl` + `weekday_evening` → 심야 운영 클래스 제외, 온라인·홈 취미 우선.

## 5️⃣ 예산 *(핵심)*

**질문 문구**: "한 달에 이 정도까지는 써볼 수 있어요"
**UI 타입**: 단일 선택 (4개 카드, 금액 명시)

| 클릭 옵션 | 내부 태그 | 벡터 값 |
|---|---|---|
| 0원 — 무료로만 | `budget_0` | budget = 0 |
| 1만원대 | `budget_low` | budget = 1 |
| 3~5만원 | `budget_mid` | budget = 2 |
| 10만원 이상 — 장비도 괜찮아요 | `budget_high` | budget = 3 |

**추천 영향**: 비용 필터·예상비용 안내. RAG 지식베이스의 취미별 예상 월비용과 직접 매칭.

## 6️⃣ 동기(목적) *(보강)*

**질문 문구**: "취미를 통해 얻고 싶은 게 뭐예요?" (최대 2개 선택)
**UI 타입**: 다중 선택 칩 (최대 2개 제한, 3번째 선택 시 기존 1개 자동 해제 or 경고)

| 클릭 옵션 | 내부 태그 |
|---|---|
| 😌 스트레스 해소 | `motive_release` |
| 🏆 성취감 | `motive_achievement` |
| 🤝 새로운 관계 | `motive_relationship` |
| 🎨 창작의 즐거움 | `motive_creation` |
| 📈 자기계발/성장 | `motive_growth` |
| 😄 그냥 재미로 | `motive_fun` |

**추천 영향**: 취미의 '이유' 매칭. 취미 DNA 리포트의 설명 문구("당신은 관계와 창작을 원해서 → OO를 골랐어요")에 직접 사용.

## 9️⃣ 기질·신규성 *(보강)*

**질문 문구**: "새로운 도전 vs 익숙한 편안함, 어느 쪽이세요?"
**UI 타입**: 5점 클릭 스케일

| 클릭 옵션 | 내부 태그 | 벡터 값 |
|---|---|---|
| 완전 새로운 도전이 좋아요 | `novelty_high` | novelty = +2 |
| 약간 낯선 것도 좋아요 | `novelty_mild` | novelty = +1 |
| 상관없어요 | `neutral` | novelty = 0 |
| 익숙한 게 편해요 | `familiar_mild` | novelty = -1 |

**추천 영향**: 탐색-활용 균형(필터버블 방지)의 기준값으로 사용. `novelty_high`일수록 세렌디피티 추천 비율 ↑.

## 🎨 UI 톤 가이드

- 문진표가 아니라 "오늘 당신의 마음을 진단해드려요"의 감성 흐름 유지
- 각 질문 상단에 진행률 표시 (1단계 5문항 / 2단계 4문항)
- 1단계 완료 시점에 "지금까지 정보로도 1차 추천이 가능해요 — 더 정확하게 찾아볼까요?" CTA로 2단계 유도 (강제 아님)
- 모든 옵션은 버튼/카드 형태, 텍스트 입력 없음 (타이핑 이탈 방지)
